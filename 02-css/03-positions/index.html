Define a Flexbox

A Flexbox is container that allows you to arrange items on a page in a flexible way.

Flexbox Properties

The Flex container properties are:

  • Flex-direction
  • Flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

The Flex-Direction property

The Flex-direction property defines in which direction the container wants to stack the flex items.
